# WebDevelopment-FinalProject

## Project Description
A Final project for a Web Development course. A multi-page portfolio documenting lessons and activities across HTML, CSS, JavaScript, Bootstrap 5, and version control with Git &amp; GitHub.

## Pages
- **Home** — Dashboard overview of all topics
- **HTML** — Lessons on structure and semantic elements
- **CSS** — Styling, layouts, Flexbox, Grid, and responsiveness
- **JavaScript** — DOM manipulation and interactivity
- **Bootstrap** — Responsive grids and UI components
- **Git & GitHub** — Version control and deployment
- **Projects** — All laboratory activities and tasks
- **Contact** — Contact form with JavaScript validation

## Features
- Dark / Light mode toggle with saved preference
- Fully responsive design (desktop, tablet, mobile)
- Embedded lesson PDFs per topic
- Live JavaScript DOM manipulation demo
- Bootstrap 5 interactive components demo
- Client-side form validation

## Technologies Used
- **Markup & Styling:** HTML, CSS
- **Scripting:** JavaScript (Vanilla)
- **Framework:** Bootstrap 5
- **Version Control:** Git & GitHub Pages

## Installation / Setup
1. Clone or download the repository
2. Open `index.html` in your browser
3. No installations or dependencies required
4. - If can't, Open this URL: https://rhem8.github.io/WebDevelopment-FinalProject/

## Usage Instructions
- Navigate between topic pages in **Home** or by using the top navbar 
- Toggle dark/light mode using the button on the top right
- Visit the **Projects** page to view all activities
- Visit the **JavaScript** page to interact with the live DOM demo
- Visit the **Bootstrap** page to interact with the live components demo
- Fill out the **Contact** form to see client-side validation in action
