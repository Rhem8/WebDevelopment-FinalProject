Bootstrap Framework

This section covers Bootstrap 5 — a popular CSS framework that helps you build responsive, mobile-first websites faster using ready-made components and a 12-column grid system.

Introduction to Bootstrap

The first lesson covers the core concepts, installation, and layout tools.

Topics included:

  • What is Bootstrap? (History & core features)
  • Three installation methods: CDN, npm, and local download
  • The 12-Column Grid System
  • Responsive breakpoints (xs, sm, md, lg, xl, xxl)
  • Advantages and trade-offs of using a CSS framework

The PDF below covers the introduction, setup, and grid layout system.

Sample Bootstrap Grid Layout

<div class="container">
    <div class="row">
        <div class="col-md-8">
            <h2>Main Content</h2>
            <p>Takes up 8 of 12 columns.</p>
        </div>
        <div class="col-md-4">
            <h2>Sidebar</h2>
            <p>Takes up 4 of 12 columns.</p>
        </div>
    </div>
</div>

The grid automatically adjusts columns based on screen size using Bootstrap's breakpoint classes.

Bootstrap Components & JavaScript

The second lesson covers Bootstrap's built-in UI components and JS integrations.

Key topics:

  • Core UI: Buttons, Cards, and responsive Navbars
  • Forms & Tables: Styled inputs and data tables
  • Alerts & Badges: Contextual notification styles
  • Dynamic Components: Modals, Carousels, and Accordions
  • Utilities: Flexbox, margin, padding, and text-alignment classes
  • JS API: Programmatic control via bootstrap.Modal, etc.

Lesson PDF 2: Bootstrap Components & JavaScript Integration

Sample Bootstrap JavaScript Usage

<button id="launchModal" class="btn btn-primary">Open Modal</button>

<script>
    const btn = document.querySelector('#launchModal');

    btn.addEventListener('click', function () {
        const modal = new bootstrap.Modal('#systemControlModal');
        modal.show();
    });
</script>

This shows how Bootstrap components can be controlled with plain JavaScript.
