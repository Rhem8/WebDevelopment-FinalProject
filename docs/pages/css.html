CSS & Responsive Design

This section covers Cascading Style Sheets (CSS) — the language used to style HTML pages. CSS controls colors, spacing, fonts, layouts, and how a website looks on different screen sizes.

Overview of CSS

CSS (Cascading Style Sheets) describes the visual presentation of HTML documents.

Sample CSS Code

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    margin: 0;
    padding: 0;
}

h1 {
    color: navy;
    text-align: center;
}

.container {
    display: flex;
    justify-content: center;
    gap: 20px;
}

This example shows basic CSS styling including colors, spacing, typography, and Flexbox layout.

Responsive Design

Responsive Web Design makes webpages adapt to different screen sizes — phones, tablets, and desktops.

  • Uses flexible layouts and grids
  • Applies media queries for screen adjustments
  • Improves accessibility and user experience
  • Supports mobile-first development
@media (min-width: 768px) {
    .container {
        display: grid;
        grid-template-columns: 1fr 1fr;
    }
}
