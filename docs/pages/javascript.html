Topics covered:

  • Variables and data types

  • Loops and iteration
  • Functions and scope

The PDF below contains the full lesson, code examples, and presentation materials.

Lesson PDF 1: JavaScript Fundamentals

Sample JavaScript Code

const student = "Maria";
let score = 85;

if (score >= 75) {
    console.log(student + " passed!");
} else {
    console.log(student + " failed.");
}

This example shows variables, conditionals, and basic JavaScript syntax.

DOM Manipulation & Interactivity

The second lesson focuses on making webpages interactive using JavaScript and the Document Object Model (DOM).

  • Selecting HTML elements
  • Changing content dynamically
  • Handling events and user interaction
  • Updating styles with JavaScript
  • Using addEventListener()

DOM manipulation lets pages become dynamic without reloading.

Lesson PDF 2: JavaScript Interactivity & DOM Manipulation

Sample DOM Manipulation

const button = document.querySelector(".btn");

button.addEventListener("click", function () {
    document.body.style.backgroundColor = "lightblue";
});

This example changes the background color of the page when the button is clicked.
