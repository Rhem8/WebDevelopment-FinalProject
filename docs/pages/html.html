HTML Topics

This section covers HTML (HyperText Markup Language) — the foundation of web development. HTML uses tags and elements to structure the content that appears on every webpage.

Introduction & Foundational Concepts

HyperText Markup Language (HTML) was created by Tim Berners-Lee in 1990. It is the standard language used to structure content on the World Wide Web.

The Core Definitions

HTML combines two key ideas:

  • HyperText: Links that connect web pages to each other.
  • Markup Language: Tags that tell the browser how to display content.

Basic HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Page</title>
</head>
<body>
    <h1>Main Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Every HTML file starts with this basic structure. The <head> holds settings, and the <body> holds visible content.
